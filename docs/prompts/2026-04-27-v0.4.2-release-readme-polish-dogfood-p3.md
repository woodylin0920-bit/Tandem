### Step 4: bootstrap.sh — dogfood P3 fix + verify P2

(a) **P3 fix**：line 211-214 的 `→ diff: diff $short_d $short_s` 印出長路徑難讀。改成 label-based 多行版：

當前：
```bash
short_s=$(echo "$s" | sed "s|$HOME|~|")
short_d=$(echo "$d" | sed "s|$HOME|~|")
echo "  $rel"
echo "    → diff: diff $short_d $short_s"
```

改成：
```bash
short_s=$(echo "$s" | sed "s|$HOME|~|")
short_d=$(echo "$d" | sed "s|$HOME|~|")
echo "  $rel"
echo "    → target:   $short_d"
echo "    → upstream: $short_s"
echo "    → diff: diff <target> <upstream>  (substitute the paths above)"
```

(b) **P2 verify**：跑一次 dry-run + apply，看 overwrite=0 時 summary 是否會錯印「1 file written」。建一個 throwaway target，把 settings.json 動一下逼出 merge-only 場景：

```bash
TMPCHK=$(mktemp -d)
bash ~/Desktop/repo/public/Tandem/bootstrap.sh "$TMPCHK/p2-verify"
cd "$TMPCHK/p2-verify"
# 把 settings.json 加一個 user-only 條目，逼出「merge needed but no overwrite」狀態
jq '.permissions.allow += ["Bash(jq:*)"]' .claude/settings.json > /tmp/p2-tmp.json && mv /tmp/p2-tmp.json .claude/settings.json
git add -A; git commit -q -m "p2 verify setup"

# 跑 upgrade --apply（其他檔都 up-to-date 應該）
output=$(bash ~/Desktop/repo/public/Tandem/bootstrap.sh --upgrade-existing "$PWD" --apply 2>&1)
echo "$output"
echo "---"

# 抓 Done. 那行
done_line=$(echo "$output" | grep "^Done\.")
echo "Done line: $done_line"

# 期望：「Done. settings.json merged.」（不該有 "1 files written"）
echo "$done_line" | grep -q "files? written" && echo "P2 STILL REPRODUCIBLE — flag in archive Result block" || echo "PASS: P2 fixed (or never reproduced) — only settings.json merged shown"

# Cleanup
cd ~/Desktop/repo/public/Tandem
SLUG_TS=$(echo "$TMPCHK/p2-verify" | sed 's|/|-|g')
rm -rf "$TMPCHK" "$HOME/.claude-work/projects/$SLUG_TS"
```

不論 P2 結果如何，**都繼續往下走**（P2 是 cosmetic，不是 blocker）。Result block 記下狀況。
